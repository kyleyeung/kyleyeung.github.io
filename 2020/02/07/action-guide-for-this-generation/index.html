这一代人的精神行动指南

Feb 7, 2020

If not now, when?

If not us, who?

注：这是一份开源的行动指南，您的建议和评论会使它变得更具有意义。来信请发送至 hi@kyleyang.net

如果您期待一个更好的世界，就应当有所行动。首先可以做的，就是寻求精神上的强健。以下是若干能够改善颓弱精神之行动，仅供参考：

  1. 停止自我审查

    永远都要问：「为什么不可以？」

    如果您想到一个问题，却担忧过于「敏感」而无法在网络上公开讨论，请不要放弃那个念头，而是妥善地选择「说」的方法（可参考下一条）。如果您实在无法想到任何一个「敏感」的话题，那您更需要思考的是：究竟是什么让我变得如此不「敏感」？

  2. 挑选聊天软件

    请不要再依赖微博和微信。某人就某事发表了一整套论述，其他用户纷纷转发评论，一旦原内容被封禁，其他人便瞬间失去了讨论的对象，从此再无继续探讨的空间。建议您使用诸如 Telegram、Signal 这样的加密通讯软件和朋友保持联系。如果您是苹果手机用户，您也可以使用 iMessage 。麻烦是必要的，微博和微信在注册使用上极其简便，却在精神上带来了极大的困难。没有人要求您必须卸载微博或微信，但至少可以做到不以它们为主，比如首先和亲密的人转移到其他平台。

    如果感到无论如何也没有办法在网络上安全地发起讨论，请约您想一同讨论的对象在线下见面，以吃饭或喝茶的名义。

  3. 重新学习中文

    是的，我们的语言塑造了我们的世界。请不要使用诸如「真香」、「阿中哥哥」和「正能量」这样的网络用语，而要使用这些用语原本所指之物的本名称呼。如果原本所指并非一个名词，请假装没有见过这个网络用语，然后尝试用语句来描述它。光复中文。

  4. 远离犬儒之人

    「趁这个机会好好提高自己」、「该干嘛干嘛」等，都是犬儒之人的常用语句。谈及一切必提「数据驱动」也是潜在的症状之一。请远离这样的人，因为他们没有让这个世界变得更好。
