## 1.flag在哪？
打开网址，BurpSuire抓包发现Cookie:
```http
Cookie:flag=suctf%7BThi5_i5_a_baby_w3b%7D
```

## 2.编码
打开网址，有输入框，submit按钮被disable掉。抓包发现Password:
```
Password: VmxST1ZtVlZNVFpVVkRBOQ==
```
用Base64多次解密后得到Su233.由于submit被disable，直接把Su233作为参数password在url里提交即可：
```
url?password=Su233
```
另一个方法是使用Chrome修改网页使按钮变得可用，方法待查。

## 3.XSS1
提交后过滤了script串，所以不能用```<script>```标签
闭合源代码中的```<pre>```标签，利用onerror属性：
```plain
</pre><img src=# onerror=alert(1)>
```

## 4.PHP是世界上最好的语言
查看源代码：
```php
if(isset($_GET["password"]) && md5($_GET["password"]) == "0")
    echo file_get_contents("/opt/flag.txt");
else
    echo file_get_contents("xedni.php");
```
此处利用了PHP语言md5()函数的缺陷：所有hash后为0e开头的串均会被解释为0，因而任意两个hash后为0e开头的串相等，这就是php的Magic Hash漏洞。
参考链接：https://www.whitehatsec.com/blog/magic-hashes/
